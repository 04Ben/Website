Allgemeine Infos über Minecraft


Spielweise

Minecraft ist ein großartiges Spiel. Ihre Möglichkeiten sind nahezu unbegrenzt.

Es gibt im großen und ganzen zwei verschiedene Spieltypen: Zum einen den Überlebensmodus (survival) und zum anderen den Kreativmodus (creativ). Im Überlebensmodus kämpft man um das nackte Überleben und versucht sich im Laufe des Spieles bessere Rüstungen, Waffen und Gegenstände zu erstellen. Auch kann man im Laufe des Spiels automatisierte Farmen bauen, welche einem das "Erfarmen" gewisser Gegenstände erleichtern. Im Kreativmodus hingegen kann man wie der Name schon sagt, seinem kreativen Geist freie Bahn lassen und mit unbegrenzten Ressourcen das Bauen, nachdem einem das Herz begehrt, wie etwa eine mittelaltertümliche Burg oder einen modernen Wolkenkratzer.

Das Ziel des Spieles ist es, am Ende in das sogenannte "End" zu gehen und den Enderdrachen zu besiegen.


Funktionsweise

Das Spiel ist an sich simpel aufgebaut, kann aber noch deutlich anspruchsvoller werden, wenn man komplexe Schaltungen mit Redstone baut.


Versionen

Es gibt für Minecraft zwei unterschiedliche Versionen. Die Java Edition und die Bedrock Edition. Sie ähneln sich sehr, so gibt es aber auch einige kleine Unterschiede. Beide Editionen wurden anfangs unabhängig von einander programmiert. Die Bedrock Edition wurde zudem ursprünglich für die mobilen Geräte programmiert, wo diese heutzutage auch noch im Einsatz ist. Sie ist aber auch auf den Konsolen vertreten und mittlerweile sogar auf dem PC. Die Java Edition hingegen gibt es nur auf dem PC. Es gibt spieltechnisch auch einige kleinere Unterschiede, diese sind aber nicht nennents Wert, da das Spielprinzip von beiden gleich ist. Am Ende ist noch mitzuteilen, dass die "Bedrock Edition" so gar nicht vom Namen her exisitiert, sondern sie nur von der Community so gennant wird, hängen von der jeweiligen Plattform ab, zum Beispiel Windows 10 Edition.

Singelplayer


Allgemeines

Im Singelplayer spawnen Sie auf einer zufällig generierten Welt. Sie fangen an Ressourcen zu farmen bis die Nacht hereinbricht. Während der Nacht können "hostile mobs" (feindliche Mobs) spawnen, welche Sie angreifen und töten wollen. Es gibt auf der anderen Seite natürlich auch noch friedliche Mobs, wie zum Beispiel "Villager", diese findet man in Dörfern. Mit diesen kann man Handel betreiben. Neben Dörfern gibt es noch weit aus mehr zufällig generierte Gebilde, wie etwa Festungen.


Welttypen

Bei der Erstellung von einer neuen Welt haben Sie noch die Möglichkeit, einen Welttypen festzulegen. So können Sie beispielsweise eine Welt generieren lassen, welche komplett flach ist oder nur aus Inseln besteht.


Seeds

Seeds sind zufällig generierte Schlüssel, welche man benötigt um eine zufällig generierte Minecraft Welt zu erstellen. Außerdem kann man den gleichen Seed nochmals eingebe,n um dieselbe Welt zu erhalten.

Multiplayer


Allgemeines

Im Multiplayer können Sie mit Ihren Freunden oder mit anderen Menschen weltweit zusammen spielen. Sie können nicht nur das normale Minecraft-Prinzip spielen, sondern auch customisierte Spielmodi, wie zum Beispiel Bedwars.


Crossplay (möglich?)

Crossplay ist möglich, wird aber nicht von allen Servern unterstützt.


Minecraft Server

Es gibt zwei Arten von Minecraft Servern. Zum einen die öffentliche Server und zum anderen private Server. Öffentliche Server sind Server, welche für jeden zugänglich sind und auf denen Sie mit zufällig ausgewählten Leuten und Freunden spielen können. Private Server, sind Server welche nur von ausgewählten Spielern betreten werden können.
