Server erstellen

Vorraussetzungen

  • Besitzen Sie das Spiel Minecraft
  • PC mit genügend Leistung
  • Java installiert (Java)
  • OpenJDK (OpenJDK)
  • Dateienerweiterung aktiv

Installation

  1. Server auf der offizielen Minecraft Seite herunterladen
  2. Ordner anlegen wo Sie Ihren Server installieren wollen
  3. Erstellen einer start.bat Datei mit dem Text: java -Xmx(Ihr maximal Wert)M -Xms(Ihr minimal Wert)M -jar minecraft_server.1.18.jar nogui
  4. starten der start.bat Datei
  5. eula von false auf trrue setzhen
  6. start.bat erneut ausführen
  7. Geben Sie in die Konsole stop ein um den Server wieder zu stoppen
  8. Server sollte nun so aussehen wie Rechts im Bild

Konfigurieren

  1. Öffnen Sie die server.probeties
  2. Ändern Sie den gewünschten Inhalt
  3. Speichern Sie das ganze ab

Starten und Loslegen

  1. Starten Sie start.bat erneut
  2. Öffnen Sie Ihr Minecraft
  3. Gehen Sie auf Multiplayer
  4. Fügen Sie einen neuen Server hinzu
  5. Geben Sie dem Server einen Namen
  6. Geben Sie als IP-Adresse localhst ein
  7. Joinen Sie dem Server
  8. Herzlichen Glückwunsche Sie sind fertig!
